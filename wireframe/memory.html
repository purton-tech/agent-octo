Hey Airbus uses memory to reduce repetition, preserve useful context, and make agent responses more relevant over time in a multi-user workspace.

How memory works

Memory is layered. Some memories belong only to a single person, some are shared with a team, and some are attached to a workspace, app, or database created by the assistant.
